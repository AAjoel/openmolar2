The Client Application
=========================

:Author: Neil Wallace (rowinggolfer AT googlemail.com)
:Release: |release|
:Date: |today|

This is the application designed for end-users allowing

-  accessing, editing and adding data to the database.
-  note keeping and treatment planning
-  generates correspondence for patients 
    
    -  appointment cards
    -  invoicing
    -  referral letters
    -  recall
